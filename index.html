Machine Learning for Stock Options Pricing

Group A - AML Final Project

Group Members: Michael Brick, Meng Li, Sharath Mohan Kumar

Project Summary

This project applies machine learning methods to predict SPY ETF maximum prices and estimate call option fair values. Unlike traditional models (e.g., Black-Scholes) that focus on expiration-day prices, we adopt a maximum likelihood frameworkâ€”predicting the highest price the underlying is likely to reach before expiration, which better reflects actual trading behavior.

Methods Used

Gradient Boosting Machine (GBM)

Multi-Layer Perceptron (MLP)
LSTM Neural Network
Black-Scholes Model

Techniques Beyond Class Coverage

  • Maximum likelihood framework for options pricing (novel application)
  • Grid search for tuning the hyperparameter in LightGBM
